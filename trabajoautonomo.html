EJERCICIOS DE TAREA

Objetivo: Realizar los ejercicios propuestos utilizando el lenguaje C#

Ejercicio 1

Escriba un programa que a partir de tres valores enteros pedidos al usuario calcule su suma y su media

Ejercicio 2

Escriba un programa que calcule las raíces de una ecuación de segundo grado cuyos coeficientes son a, b y c. Utilice el método matemático Sqrt de la clase Math del espacio de nombres System, que calcula la raíz cuadrada de un número que no puede ser negativo y aplicable sobre un tipo de dato double

Ejercicio 3

Escriba un programa que represente una pequeña calculadora. Para ello, el programa pedirá dos números enteros y una operación aritmética (que debe ser +, -, * o /) y mostrará el resultado correspondiente

Ejercicio 4

Escriba un programa que verifique si un número introducido por el usuario es múltiplo de 5. Utilizar para ello el operador % (módulo), que devuelve el resto de la división entre dos números enteros

Ejercicio 5

Escriba un programa que calcule la factorial de un número entero positivo N introducido por el usuario. Obtenga dos versiones, una utilizando la sentencia while y otra usando la sentencia for, para compararlas

Ejercicio 6

Escriba un programa que le pida al usuario su nombre y una clave. A continuación, le dirá que la clave ha caducado y que tiene que cambiarla, mostrando el mensaje “Su clave ha caducado. Por favor, introduzca su nueva clave”. Se debe confirmar la clave y permitir un máximo de 5 intentos.

Ejercicio 7

Cree un programa para operar con números complejos. El programa estará formado por tres clases diferentes: principal, Numero Complejo y Aritmética Compleja. Las funciones incluirán: suma, resta, multiplicación, división, conversión de forma binomial a polar y viceversa.

Ejercicio 8

Escriba un programa para la gestión de las calificaciones de un grupo de alumnos (DNI, Apellidos, Nombre, Nota, Calificación). Incluye funcionalidades como introducir, eliminar, consultar, modificar y listar alumnos.

Ejercicio 9

Desarrolle un programa para la gestión de un videoclub que ofrece tanto películas como videojuegos en alquiler. Llevar registro de películas y videojuegos, marcar alquileres y controlar el número de veces que se ha alquilado cada uno.
